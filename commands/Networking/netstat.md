# netstat

- Requires the package `net-tools`

```bash
# Print network connections
netstat
netstat -rnv # Show the gateways

# Print all ports currently open and listening
netstat -tunlp
  # listening sockets (-l)
  # port number (-n)
  # TCP ports (-t)
  # UDP ports (-u)
  # display PID/Program name for sockets


```
